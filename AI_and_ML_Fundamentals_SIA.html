Slide 1 • AI & Machine Learning Fundamentals

What is AI?

Artificial Intelligence (AI) is when machines perform tasks that typically require human intelligence — for example recognizing speech, making decisions, or learning from experience.

  • Chatbots that answer questions
  • Recommendation systems (movies, music)
  • Self-driving car features and computer vision
Slide 2 • What is AI?

Types of AI

There are two commonly discussed categories:

  • Narrow AI — Designed for specific tasks (e.g., voice assistants, spam filters).
  • General AI — A theoretical form of AI with human-like flexible intelligence (not yet achieved).
Slide 3 • Types of AI

What is Machine Learning?

Machine Learning (ML) is a subset of AI that teaches computers to learn patterns from data so they can make predictions or decisions without being explicitly programmed for that specific task.

Example: An ML model can learn to tell whether an email is spam by training on many labeled examples.

Slide 4 • What is Machine Learning?

Types of Machine Learning

  • Supervised Learning — Trained on labeled data (input → correct output).
  • Unsupervised Learning — Finds structure in unlabeled data (e.g., clusters).
  • Reinforcement Learning — Learns through trial and error with feedback (rewards).
Slide 5 • Types of ML

How Machine Learning Works

  1. Collect data
  2. Train a model (algorithm learns from data)
  3. Make predictions on new inputs
  4. Evaluate performance and improve the model

ML uses algorithms and data to identify patterns and make predictions with increasing accuracy over time.

Real-world Applications

  • Healthcare: Diagnosis support, patient monitoring
  • Education: Personalized learning and tutoring
  • Entertainment: Recommendations for movies & music
  • Finance: Fraud detection, algorithmic trading
Slide 7 • Real-world Applications

Key Takeaways

• AI helps machines perform tasks that look like human intelligence.
• Machine Learning is how machines learn from data to make better predictions.
• AI and ML are already part of many everyday products and will shape future innovation.
